# 泛型

> 《Rust 程序设计语言》：[泛型](https://doc.rust-lang.org/book/ch10-01-syntax.html)。

泛型让函数、struct、enum 和方法可以复用同一套逻辑，同时保留 Rust 的静态类型检查与零成本抽象。
